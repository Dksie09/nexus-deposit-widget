# Nexus Deposit Widget

A simple, modern and animated widget for Avail Nexus. [live link](https://nexus-deposit.vercel.app/)

## Demo

https://github.com/user-attachments/assets/81dd3119-7ff6-4c0c-bf8f-2057fdabf85f

## Tech Stack

- Next.js
- TypeScript
- TailwindCSS v4
- Motion
- Shadcn/ui: button/ input/ select
- Lucide React

## File Structure

```
nexus-widget-ui/
├── app/                  # app directory
│   ├── layout.tsx        # Root layout
│   ├── page.tsx          # Home page
│   └── globals.css       # Global styles
├── components/           # React components
│   ├── ui/               # Reusable UI components
│   ├── shared/           # Shared components
│   └── *.tsx             # Feature components
├── hooks/                # Custom React hooks
├── utils/                # Utility functions
├── constants/            # App constants
└── lib/                  # Library configurations
```

## Local Setup


   ```bash
   git clone https://github.com/Dksie09/nexus-deposit-widget
   cd nexus-widget-ui
   npm install
   npm run dev
   ```

**Open in browser**
   ```
   http://localhost:3000
   ```
